# Komal Gaikwad — Portfolio

Personal portfolio website — Cloud Administrator (AWS) profile, experience, projects, and certifications.

**Live site:** enable GitHub Pages on this repo (Settings → Pages → Deploy from branch `main` / root):
`https://komalgithubofficial.github.io/portfolio/`

## Stack
- HTML5, CSS3 (no frameworks, no build step)
- Vanilla JS (scroll-reveal, contact form via mailto)

## Structure
```
index.html   → the entire site
resume.pdf   → downloadable resume (linked from the nav "Resume" button)
```

## Updating
- Site content: edit `index.html` — sections run top to bottom (Hero, Tech Stack, Experience, Projects, Certifications, Contact).
- Certifications: once the AWS Certified Cloud Practitioner exam is passed, update the `certifications` section's status badge from "Exam Scheduled" to "Certified" and add the credential link.
- New projects: add a `.proj-card` block inside `#projects` following the existing pattern.
- Resume: replace `resume.pdf` with an updated export any time the resume changes (keep the filename the same so the download button keeps working).
